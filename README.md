# Sample of backend application with micronaut

## Build

```shell
./gradlew dockerBuildNative
```
note: <br>
If you get a error with code 137, retry build your app.
<br>
This code meant for you that you didn't have enough memory available.


## Run
```shell
docker compose up -d
```
